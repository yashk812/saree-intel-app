# 🥻 Saree Intel — Kalyan Silks Competitive Dashboard

Competitive intelligence dashboard tracking 235 stores across 6 saree retail companies.

## Companies tracked
- **Kalyan Silks** (target) — 27 stores
- SSKL Ltd (Kalamandir, KLM Fashion Mall, Brand Mandir, Kancheepuram Varamahalakshmi Silks) — 70 stores
- RS Brothers (RS Brothers, South India Shopping Mall) — 57 stores
- Marri Retail (The Chennai Shopping Mall, JC Mall, JC Brothers) — 31 stores
- Nalli — 31 stores
- Pothys — 19 stores

## Pages
1. **Store Map** — Interactive map with clustering, filters by state/district/city
2. **Company Stats** — KPI cards, bar charts, city-level heatmap
3. **Expansion Insights** — White-space cities ranked by opportunity score, opportunity map
4. **Master Data** — Full searchable table with CSV download

## Running locally
```bash
pip install -r requirements.txt
streamlit run app.py
```
